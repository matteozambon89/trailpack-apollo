GraphQL for Trails.JS powered by Apollo

## Install

```sh
$ npm install --save trailpack-apollo
```

## Configure

```js
// config/main.js
module.exports = {
  packs: [
    // ... other trailpacks
    require('trailpack-apollo')
  ]
}
```
